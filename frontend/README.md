# OVGS - Frontend

Interface do Sistema de Gestão de Ordens de Venda (Next.js + Tailwind).

A documentação completa (divisão de estado, decisões e trade-offs) está no
[README raiz](../README.md).

## Scripts

```bash
pnpm dev     # desenvolvimento
pnpm build   # build de produção
pnpm test    # testes unitários (Vitest)
pnpm lint    # eslint
```

Requer a API no ar. A URL é configurada por `NEXT_PUBLIC_API_URL`
(padrão `http://localhost:3333`).
